# Ecommerce
Bhavi India Fashion is an eCommerce platform for limited traditional fashion products (Kurtis, Pants, Cholis, etc.). Targeted at retailers, wholesalers, and individual customers for single/bulk orders.
Key Features:

Product Catalog: Browse & purchase single/bulk items.
Custom Tailoring: Users select fabric, enter sizes, choose tailor; admin approves/rejects.
Virtual Try-On: Basic image overlay (upload photo + product overlay).
Admin Panel: Order/design approval, tailor management, inventory control.
Payments: COD, Net Banking, UPI, Cards, Wallets (Razorpay/Paytm integration).
Chatbot: Real-time customer query support.
Tailor Workflow: Tailors receive jobs, submit completed work for admin approval.
Backend: Django + MySQL (user auth, orders, products, designs, payments).
Tech Stack: HTML/CSS/JS/Bootstrap/jQuery (Frontend), Django (Backend), MySQL (DB), AWS/Heroku hosting, SSL security.
